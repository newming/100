CSS3 background-size 属性

日期：2016.1.12

Learn more »

实例

规定背景图像的尺寸：

div
{
background:url(img_flwr.gif);
background-size:80px 60px;
background-repeat:no-repeat;
}

语法

background-size: length|percentage|cover|contain;
值 描述 个人说明
length 设置背景图像的高度和宽度。第一个值设置宽度，第二个值设置高度。如果只设置一个值，则第二个值会被设置为 "auto"。 给予背景图固定大小，px像素单位，往常是只能给图片调整大小的。
percentage 以父元素的百分比来设置背景图像的宽度和高度。第一个值设置宽度，第二个值设置高度。如果只设置一个值，则第二个值会被设置为 "auto"。 这种用法是给予百分数大小，可以随窗口大小改变，如同流媒体图片。可能使图像变形。
cover 把背景图像扩展至足够大，以使背景图像完全覆盖背景区域。背景图像的某些部分也许无法显示在背景定位区域中。 等比例放大图像，不会使图像变形，又可以完全覆盖背景区域，可以配合background-position控制由何处缩放。
contain 把图像图像扩展至最大尺寸，以使其宽度或高度完全适应内容区域。 这个属性是等比例放大或缩小图像，使宽度或高度一方达到适应内容区域。

以下四个分别为四种情况，内容区域（盒子大小为width90%，height500px）

这里设置了宽度90%，高度auto

cover,比较好用，总是铺满内容区域，此处配合background-position：50% 50%;使图片从中心缩放。
